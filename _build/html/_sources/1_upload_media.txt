1. Upload Media
===============
{IMAGE OF UPLOAD SCREEN}

Uploading a file is he first stage in preparing it for download. Once the file has been uploaded, it can be used in the Pixel Strings service.

Alternatively, if you already have an Amazon Cloud account with your desired footage, you can link to that bucket directly - this is far faster than uploading footage through the web interface.
